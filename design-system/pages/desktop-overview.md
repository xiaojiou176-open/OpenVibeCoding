# Desktop Overview Override

Desktop should feel like the **focused command deck**.

## Primary job

Surface the same command-tower loop as the web front door, but with denser,
faster operator scanning.

## Rules

- Keep the main loop visually obvious.
- Let quick actions feel immediate and tool-grade.
- Desktop can be denser than web, but not noisier.

## Avoid

- mobile-consumer animation language
- fake haptics metaphors
- generic marketing hero layout
