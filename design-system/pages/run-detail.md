# Run Detail Override

This page is the **truth room**.

## Primary job

Tell the operator:

- what changed
- what passed or failed
- what evidence exists
- what next action is safe

## Rules

- Evidence and replay affordances should appear early.
- Contract and binding summaries should remain visible without competing with
  incident/proof context.
- Run detail should feel operational and forensic, not event-marketing.

## Avoid

- conference/event page structure
- sponsor/speaker style hierarchy
- decorative urgency gimmicks
