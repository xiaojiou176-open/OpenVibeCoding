# Command Tower Override

This page should feel like the active **L0 cockpit**.

## Primary job

Answer three questions immediately:

1. What is happening right now?
2. What is risky?
3. What should the operator do next?

## Rules

- Risk summary must be visible above the fold.
- Primary actions should bias toward recovery, focus, and truth inspection.
- Context drawer should feel like an instrument panel, not a side note.
- Use density, but preserve calm scan order.

## Avoid

- generic “monitoring dashboard” vibe
- overuse of decorative status colors
- equal-weight sections that hide priority
